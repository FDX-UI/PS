1. 基本概念
术语 描述
命令行 在 PowerShell 中输入命令的地方
命令 执行特定操作的语句，如 Get-Process
参数 命令的附加选项，通常以 - 开头，如 -Name
参数值 参数的具体值，如 -Name "notepad"
别名/简写 命令的简写形式，如 ls 是 Get-ChildItem 的别名
对象 PowerShell 中的数据结构，命令的输出通常是一个对象

2. 常用命令
命令 功能 常用参数 示例
Get-Command 列出所有可用命令或按名称、模块等过滤命令 -Name <名称>、-Module <模块名> Get-Command -Name Get-*
Get-Help 获取命令的帮助信息，支持详细帮助、示例等 -Name <命令>、-Detailed、-Examples Get-Help Get-Process -Detailed
Get-Process 获取系统进程 -Name <进程名> Get-Process -Name notepad
Stop-Process 终止进程 -Id <进程ID> Stop-Process -Id 1234
Get-Service 获取系统服务 -Name <服务名> Get-Service -Name Spooler
Start-Service 启动服务 -Name <服务名> Start-Service -Name Spooler
Stop-Service 停止服务 -Name <服务名> Stop-Service -Name Spooler
Get-ChildItem 列出目录内容 -Path <路径>、-Recurse Get-ChildItem -Path C:\ -Recurse
Copy-Item 复制文件或目录 -Path <源路径>、-Destination <目标路径> Copy-Item -Path C:\source.txt -Destination C:\destination.txt
Remove-Item 删除文件或目录 -Path <路径>、-Recurse Remove-Item -Path C:\OldFolder -Recurse
Rename-Item 重命名文件或目录 -Path <路径>、-NewName <新名称> Rename-Item -Path C:\OldName.txt -NewName NewName.txt
Set-Location 切换目录 -Path <路径> Set-Location -Path C:\Users
New-Item 创建文件或目录 -ItemType <类型>、-Path <路径>、-Name <名称> New-Item -ItemType Directory -Path C:\ -Name NewFolder
Invoke-WebRequest 发送 HTTP 请求 -Uri 、-Method <方法> Invoke-WebRequest -Uri https://www.example.com
Get-Content 读取文件内容 -Path <路径> Get-Content -Path C:\file.txt
Set-Content 写入内容到文件 -Path <路径>、-Value <内容> Set-Content -Path C:\file.txt -Value "Hello, World!"

3. 输出/输入
命令 功能 常用参数 示例
Write-Output 将内容输出到控制台或管道 -InputObject <对象> Write-Output "Hello, World!"
Write-Host 将内容输出到控制台，支持颜色等格式化 -ForegroundColor <颜色>、-BackgroundColor <颜色> Write-Host "Hello, World!" -ForegroundColor Green
Out-File 将输出保存到文件 -FilePath <路径>、-Append Out-File -FilePath C:\processes.txt
Out-Null 将输出丢弃，不显示在控制台 暂无 Out-Null
Read-Host 从用户获取输入 -Prompt <提示信息> $name = Read-Host "请输入您的名字"
Get-Content 从文件读取内容 -Path <路径>、-First <行数>、-Last <行数> Get-Content -Path C:\file.txt
Set-Content 将内容写入到文件 -Path <路径>、-Value <内容> Set-Content -Path C:\file.txt -Value "Hello, World!"

4. 变量
语法 描述
$variable = value 使用变量
$null 表示空值
$true 和 $false 布尔值
$env:变量名 系统环境变量，如 $env:PATH

5. 数据类型
数据类型 别名/简写 描述 示例 类型强制
String 暂无 用于存储文本数据 $str = "Hello, World!" [String] 变量名
Integer int 用于存储整数 $num = 42 [Integer] 变量名
Double 暂无 用于存储浮点数 $float = 3.14 [Double] 变量名
Boolean bool 用于存储布尔值 $isTrue = $true [Boolean] 变量名
Array 暂无 用于存储一组值 $array = @(1, 2, 3, "four") [Array] 变量名
Hashtable 暂无 用于存储键值对 $hash = @{Name = "John"; Age = 30} [Hashtable] 变量名
DateTime 暂无 用于存储日期和时间 $date = Get-Date [DateTime] 变量名
Object 暂无 用于存储复杂数据结构 $process = Get-Process [Object] 变量名
ScriptBlock 暂无 用于存储代码块 $script = { Write-Output "Hello" } [ScriptBlock] 变量名
Null 暂无 表示空值 $variableName = $null [Null] 变量名

7. 流程控制
语句类型 描述 语法结构 示例
if 根据条件执行特定代码块 if (条件) {代码块}
elseif (条件) {代码块}
else {代码块}		 if ($num -gt 10) {
Write-Output "num 大于 10"
} elseif ($num -eq 10) {
Write-Output "num 等于 10"
} else {
Write-Output "num 小于 10"
}
switch 根据表达式的值匹配并执行对应的代码块 switch (表达式) {
值1 {代码块}
值2 {代码块}
默认 {代码块}
}		 $day = "Monday"
switch ($day) {
"Monday" { Write-Output "今天是周一" }
"Tuesday" { Write-Output "今天是周二" }
default { Write-Output "其他日子" }
}
for 循环执行代码块，直到条件不满足 for (初始化; 条件; 步进) {代码块} for ($i = 1; $i -le 5; $i++) {
Write-Output $i
}
foreach 遍历集合中的每个元素并执行代码块 foreach ($元素 in $集合) {代码块} $numbers = 1, 2, 3, 4, 5
foreach ($number in $numbers) {
Write-Output $number
}
while 当条件为真时，循环执行代码块 while (条件) {代码块} $i = 1
while ($i -le 5) {
Write-Output $i
$i++
}
do 循环执行代码块，至少执行一次 do {代码块} while (条件)
do {代码块} until (条件)		 $i = 1
do {
Write-Output $i
$i++
} while ($i -le 5)

8. 函数
函数类型 描述 语法结构 示例
定义函数 创建一个可复用的代码块，用于执行特定任务 function 函数名 {
参数定义
函数体
}		 function Add-Numbers {
param ($a, $b)
return $a + $b
}
调用函数 执行函数并传递参数 函数名 参数 $result = Add-Numbers 5 3
Write-Output $result
参数(位置参数) 按位置传递参数 param ($param1, $param2) function Greet {
param ($name, $greeting)
Write-Output "$greeting, $name!"
}
Greet -name "Alice" -greeting "Hello"
默认参数值 为参数设置默认值 param ($param1 = 默认值) function Greet {
param ($name = "Guest", $greeting = "Hello")
Write-Output "$greeting, $name!"
}
动态参数 使用 CmdletBinding 添加动态参数 [CmdletBinding()]
param (
[Parameter(Mandatory = $true)]
[string]$param1
)		 [CmdletBinding()]
param (
[Parameter(Mandatory = $true)]
[string]$name
)
function Greet {
Write-Output "Hello, $name!"
}
Greet -name "Alice"
返回值 从函数返回值 return 表达式 function Multiply-Numbers {
param ($a, $b)
return $a * $b
}
$result = Multiply-Numbers 4 5
Write-Output $result
函数嵌套 在函数内部定义和调用其他函数 function 函数名 {
function 内部函数 {
...
}
...
}		 function Outer-Function {

Write-Output "Inner function called"

Inner-Function
}

9. 异常处理
功能 描述 语法结构 示例
try 包含可能会引发异常的代码 try {
可能会引发异常的代码
}		 try {
$result = 10 / 0
}
catch 捕获并处理异常 catch {
处理异常的代码
}
或
catch [异常类型] {
处理特定异常的代码
}		 try {
$result = 10 / 0
} catch {
Write-Output "发生错误: $_"
}
finally 无论是否发生异常都会执行的代码 finally {
无论是否发生异常都会执行的代码
}		 try {
$result = 10 / 0
} catch {
Write-Output "发生错误: $_"
} finally {
Write-Output "清理资源"
}
throw 手动抛出异常 throw "错误信息"
或
throw [异常对象] 		if ($x -lt 0) {
throw "x 不能小于 0"
}
捕获特定异常 捕获特定类型的异常 catch [异常类型] {
处理特定异常的代码
}		 try {
$result = 10 / 0
} catch [System.DivideByZeroException] {
Write-Output "发生除以零错误"
} catch {
Write-Output "发生其他错误: $_"
}
获取异常详细信息 获取捕获的异常对象 $_ 或 $Error try {
$result = 10 / 0
} catch {
Write-Output "错误信息: $_"
Write-Output "错误类型: $_.GetType().Name"
}
继续执行脚本 捕获异常后继续执行脚本 try-catch try {
$result = 10 / 0
} catch {
Write-Output "发生错误: $_"
}
Write-Output "脚本继续执行"
